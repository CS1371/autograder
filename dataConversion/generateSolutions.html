generateSolutions: Generate the solution values for comparison

This will generate the solution values, given a path to the solution ZIP archive. These solutions are held in a `Problem` array.

PROBLEMS = generateSolutions(PATH) will return a Problem Array containing the problems for the current homework specified by PATH, which is a string representation of the path to the solution ZIP archive

Remarks

Any exceptions thrown by the Problem class or sub classes will not be caught by generateSolutions, and will instead by propogated forward. These errors are considered mostly fatal.

The JSON format is strictly enforced. For more information on what this format should look like, see the central documentation. For convenience, an example is shown below.

Exceptions

generateSolutions throws exception AUTOGRADER:GENERATESOLUTIONS:INVALIDPATH if the input path is invalid or if necessary file is missing in the given directory
