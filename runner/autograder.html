autograder: Main responsibility for running the autograder

autograder is called by the Autograder class - it is responsible for running the actual steps needed to grade students automatically.

Remarks

Main is the entry point for the Autograder. Main will handle any exceptions, and will always clean up after itself - the user's folder will not be changed!
